Week One

HTML and CSS

The first thing we did was build a basic website that looked similar to the one shown below, using the basic html and css knowledge learned this week. To summarize it all up, HTML is the body of the website. Think of it like a car. We would say a car has wheels and an engine like a website has a side-bar, a nav bar, and a body. CSS describes what the body looks like. This would be like saying the car is red, with black tires. The website is red with a black nav bar.

My end product for this project was a basic website that had a few links, a few pictures, and some basic toolbars. During the second week I added a tic tac toe game to my information box on the left. Some of the source code can be seen here. I then began learning about bootstrap, so I dropped that old site and started this heavily bootstrap-influenced website.
